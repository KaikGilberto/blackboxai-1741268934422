1. Estrutura do Projeto

loomtech-portfolio/
├── index.html      # Página principal
├── portfolio.html  # Página de portfólio
└── doc.html       # Esta documentação

Descrição dos Arquivos

  • index.html: Página principal do site contendo:
    • Header com navegação
    • Seção Hero
    • Seção Sobre
    • Seção Serviços
    • Seção Contato
    • Footer
  • portfolio.html: Página de portfólio contendo:
    • Grid de projetos
    • Sistema de filtros
    • Cards de projetos
  • doc.html: Esta documentação completa do site

2. Tecnologias Utilizadas

  • Tailwind CSS:

    Framework CSS utilizado para estilização através de classes utilitárias. Incluído via CDN:

    <script src="https://cdn.tailwindcss.com"></script>
  • Google Fonts:

    Fontes utilizadas: Space Grotesk para títulos e Inter para texto. Incluídas via CDN:

    <link href="https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@400;500;600;700&family=Inter:wght@300;400;500;600&display=swap" rel="stylesheet">
  • Font Awesome:

    Biblioteca de ícones. Incluída via CDN:

    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0-beta3/css/all.min.css">

3. Como Fazer Modificações

Cores e Tema

As cores principais do tema são definidas no Tailwind Config de cada página: 

tailwind.config = {
    theme: {
        extend: {
            colors: {
                'primary': '#10b981',    // Cor principal
                'secondary': '#065f46',   // Cor secundária
                'dark': '#0f172a',        // Cor de fundo
            }
        }
    }
}

Textos e Conteúdo

Para modificar textos, localize a seção desejada no HTML e altere o conteúdo dentro das tags. Exemplo para alterar o título principal: 

<h1 class="text-5xl font-space font-bold mb-6 gradient-text">
    Seu novo título aqui
</h1>

Links e Navegação

Os links de navegação estão no header de cada página. Para modificar: 

<div class="hidden md:flex space-x-8">
    <a href="index.html" class="text-slate-300 hover:text-primary transition-colors">Home</a>
    <a href="portfolio.html" class="text-slate-300 hover:text-primary transition-colors">Portfólio</a>
    ...
</div>

4. Componentes do Site

Header/Navbar

O header é fixo no topo e inclui o logo e links de navegação. Para mobile, há um menu hamburguer que abre um menu fullscreen.

Seções

Cada seção segue um padrão similar: 

<section id="secao" class="py-20">
    <div class="container mx-auto px-6">
        <h2 class="text-4xl font-space font-bold mb-12 text-center gradient-text">
            Título da Seção
        </h2>
        <!-- Conteúdo da seção -->
    </div>
</section>

Cards

Os cards são utilizados no portfólio e na seção de serviços: 

<div class="p-6 bg-slate-900 rounded-lg border border-slate-800">
    <h3 class="text-xl font-space font-bold mb-2">Título do Card</h3>
    <p class="text-slate-300 mb-4">Descrição do card</p>
    <!-- Mais conteúdo -->
</div>

5. Gerenciando o Portfólio

Adicionando Novos Projetos

Para adicionar um novo projeto ao portfólio, copie a estrutura do card de projeto e modifique: 

<div class="group" data-category="categoria">
    <div class="relative overflow-hidden rounded-lg bg-slate-900 border border-slate-800">
        <div class="aspect-video bg-slate-800 flex items-center justify-center">
            <i class="fas fa-icon text-6xl text-primary/20"></i>
        </div>
        <div class="p-6">
            <h3 class="text-xl font-space font-bold mb-2">Nome do Projeto</h3>
            <p class="text-slate-300 mb-4">Descrição do projeto</p>
            <div class="flex flex-wrap gap-2 mb-4">
                <span class="px-3 py-1 bg-primary/10 text-primary rounded-full text-sm">Tecnologia 1</span>
                <span class="px-3 py-1 bg-primary/10 text-primary rounded-full text-sm">Tecnologia 2</span>
            </div>
            <a href="#" class="inline-flex items-center text-primary hover:text-primary/80 transition-colors">
                Ver Detalhes
                <i class="fas fa-arrow-right ml-2"></i>
            </a>
        </div>
    </div>
</div>

Categorias de Projetos

As categorias são definidas no atributo data-category e nos botões de filtro: 

<!-- Botão de filtro -->
<button class="px-6 py-2 rounded-full bg-dark border border-slate-800" data-filter="categoria">
    Nome da Categoria
</button>

<!-- Card do projeto -->
<div class="group" data-category="categoria">
    ...
</div>

6. Otimização e Performance

Imagens

- Use imagens otimizadas e comprimidas
- Defina width e height para evitar layout shift
- Considere usar lazy loading para imagens abaixo da dobra

JavaScript

- O JavaScript é mínimo e focado em funcionalidades essenciais
- Scripts são carregados no final do body
- Event listeners são adicionados após o DOMContentLoaded

CSS

- Tailwind CSS é carregado via CDN para desenvolvimento
- Para produção, considere compilar e purgar o CSS não utilizado
- As classes são organizadas de forma consistente
